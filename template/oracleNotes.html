Data:-Data is a row facts which describe the attributes of an entity.
Entity:-Entity is nothing but it explains both living things and non living things.
Attributes:- Attributes are also known as properties.
Database:-Database is a place or medium which is use to store the data systematic and organized manner.

In database we have some basic operations.

i. Create/Insert
ii. Read/Retrive
iii. Update/modify
iv. Delete/Drop

These are formally called CRUD  operation.

DBMS:- DBMS stands for database management system. it is a software which is use to 
mantain and manage the database.

 i. In DBMS the data are store in file formates.
 ii.  We use Query language (QL) for communicate or interact with DBMS.
 iii. . Security and autharization are the two important features provided by DBMS.

RDBMS:-RDBMS stands for relational database management system. RDBMS is a type of DBMS software.

 i.   In RDBMS the data are store in Table formates.
 ii.  We use Structured Query language (SQL) for communicate or interact with DBMS.
 iii. Security and autharization are the two important features provided by RDBMS.
 iv.  Any DBMS sofware which follows relational model is known as RDBMS.
